How BB84 Works

1. Alice (Phone) sends random bits encoded as photon polarizations over a quantum channel.

2. Bob (Server) measures each photon using a randomly chosen basis.

3. Sifting — only measurements where both chose the same basis are kept.

4. Privacy Amplification hashes the sifted key to eliminate any info an eavesdropper could have.

5. Result — both sides share an identical secret key that was never transmitted classically.
